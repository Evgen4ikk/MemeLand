# MemeLand

## Getting start

### Start JSON server
-Open Visual Studio witch project

-Open CMD(Ctrl+Shift+C)

-Install JSON server: ``` npm install -g json-server ```

-Start JSON server: ``` json-server --watch db.json --port 3000 ```




### Start web-application
```
npm i
```

```
npm run dev
```
